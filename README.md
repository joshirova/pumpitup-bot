# PumpItUp Bot — Прототип рекомендательной фитнес-системы на основе ранжирования

## О проекте

Данный проект представляет собой научно-исследовательскую работу (НИР), выполненную в рамках практики студенткой 4 курса РУДН **Ошировой Юлией Николаевной** (группа НФИбд-01-22). Проект посвящён решению ключевой проблемы в области персонализированных фитнес-рекомендаций — переходу от традиционной классификации к задаче ранжирования.

### Научная новизна

Основная научная идея проекта заключается в том, что задача фитнес-рекомендаций по своей природе является задачей **ранжирования**, а не классификации. В отличие от классических систем, которые предсказывают один «оптимальный» план тренировок, наша система возвращает **топ-3 рекомендаций** с учётом их релевантности для пользователя. Это соответствует реальной практике фитнес-приложений, где пользователю предлагается несколько вариантов тренировок на неделю (понедельник, среда, пятница).

Ключевые особенности подхода:
- **Метрика качества NDCG@3** вместо традиционных accuracy/F1-score — учитывает позицию правильного плана в топ-3 рекомендаций
- **Интерпретируемость рекомендаций** через анализ важности признаков (ИМТ, цель тренировок)
- **Прогнозирование прогресса** — оценка времени достижения целевого ИМТ на основе линейной модели
- **Соответствие клиническим рекомендациям ВОЗ** — рекомендации формируются с учётом классификации ИМТ (норма, избыточный вес, ожирение)

### Результаты исследования

Эксперименты показали, что:
- Все протестированные модели демонстрируют **идеальный Hit Rate@3 = 1.0** (правильный план всегда попадает в топ-3)
- Наилучшее качество ранжирования показала **логистическая регрессия (NDCG@3 = 0.7866)**
- Анализ важности признаков подтвердил клиническую значимость **ИМТ (66.66%)** и **цели тренировок (33.58%)**
- Простые модели (логистическая регрессия, дерево решений) превосходят ансамблевые методы в задаче ранжирования при ограниченном признаковом пространстве

## Установка и запуск

### Требования
- Python 3.9+
- pip (менеджер пакетов Python)

### Пошаговая инструкция

#### Шаг 1: Клонирование репозитория
```bash
git clone https://github.com/joshirova/pumpitup-bot.git
cd pumpitup-bot
```

#### Шаг 2: Установка зависимостей
```bash
pip install -r requirements.txt
```

#### Шаг 3: Настройка Telegram-бота
1. Создайте файл `.env` в корне проекта:
   ```bash
   cp .env.example .env
   ```
2. Получите токен бота у [@BotFather](https://t.me/BotFather) в Telegram
3. Вставьте токен в файл `.env`:
   ```env
   TELEGRAM_TOKEN=ваш_токен_бота_здесь
   ```

#### Шаг 4: Запуск бота
```bash
python bot.py
```
После запуска в консоли появится сообщение:
```
🤖 Бот запущен. Жди сообщений в Telegram!
```

#### Шаг 5: Использование бота в Telegram
1. Откройте Telegram и найдите вашего бота по имени (указано при создании через @BotFather)
2. Отправьте команду `/start`
3. Следуйте инструкциям бота:
   - Выберите цель тренировок (Похудение / Набор массы / Поддержание формы)
   - Укажите пол, возрастную группу, уровень подготовки, тип тренировки
   - Введите рост и вес для расчёта ИМТ
4. Получите топ-3 рекомендованных плана тренировок на неделю

#### Дополнительные команды
- `/explain` — объяснение рекомендаций через график важности признаков
- `/metrics` — сравнение метрик качества (ранжирование vs классификация)
- `/progress` — прогноз времени достижения целевого ИМТ с графиком динамики

## Результаты тестирования

### Сравнение моделей по метрикам ранжирования

| Модель                | NDCG@3 | Hit Rate@3 | Accuracy | F1-score |
|-----------------------|--------|------------|----------|----------|
| Логистическая регрессия | 0.7866 | 1.0        | 0.501    | 0.454    |

| XGBoost               | 0.7659 | 1.0        | 0.455    | 0.447    |
| Случайный лес         | 0.7564 | 1.0        | 0.430    | 0.421    |

**Для логистической регрессии:**

- Пол: 12.14%
